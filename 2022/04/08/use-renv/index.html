Use renv

2022-04-08

This is a blog describing how to use package renv to facilate the running of github actions.

  1. Open project, remove all the packages will be used.

  2. Library renv, run renu::init().

  3. Install necessary packages.

  4. Run renv::snapshot().

  5. Push all the files to github, done.

Pay attention to curl, it often suffers problems in github actions.
