Using testthat and covr

2022-04-16

This is a blog describing how to use testthat and covr.

  1. In your package, run usethis::use_testthat() to set up the tests folder.

  2. For each function, run usethis::use_test(“Rfuncname”). It will automatically create the corresponding test function for each R function.

  3. Add test block including multiple testing statements. e.g:

context(“test-basics”)

test_that(“multiplication works”,

{expect_equal(2 * 2, 4)

expect_output(function(x, y))

expect_length(function(x, y), 1)}

)

  1. Test packages using Ctrl/CMD - Shift - T or use the Build Menu

  2. Use covr::report() to see how much are covered.
