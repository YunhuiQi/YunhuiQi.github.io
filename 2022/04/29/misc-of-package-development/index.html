Misc of package development

2022-04-29

PAT:

When personal access token expires, generate a new one and update:

gitcreds::gitcreds_set()

usethis::edit_r_environ() and check whether the PAT is new.

Sys.getenv(“GITHUB_PAT”) to check whether the PAT is new

Badge:

license badge is just add the corresponding line to readme.

R-CMD-check is just usethis::use_github_actions()

Readme and vignettes:

Everytime update, run devtools::build_readme() and knit readme.rmd and pkgdown::build_sites()
