Create an R package Basic

2022-03-14

This is a blog describing my first experience creating a simple R package on class STAT 585 at ISU.

Basic Workflow

  1. Create a basic R package in R studio using New Project - New Dictionary - R Package

  2. Write functions in R/FuncName.R, including roxygen comments at the beginning of each function.

  3. Run devtools::document() to update NAMESPACE and man/FuncName.Rd

  4. Run devtools::check() to make sure there is no error, no warning, no notes before build the package

  5. Using CMD+shift+B to build the package, it will automatically library the new package so that we can try it ourselves.

Using other packages in FuncName.R

  1. Use package MASS: roxygen comment @import MASS, Description usethis::use_package(“MASS”,type=“import”), MASS::FunctionToUse in main of FuncName.R

  2. Library package MASS: @depends

  3. Others: @suggests

Including data

  1. Data should be in .rda file. Use usethis::use_data(newdata), it will create an newdata.rda file in data/.

  2. Manually create newdata.R in R/, including roxygen comments at the beginning, and include “newdata” as the main function text.
